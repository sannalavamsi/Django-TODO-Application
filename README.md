# Django-TODO-Application

TechStack Used: 
1. Framework: Django = 3.2.14
2. Language: Python = 3.9.6 

Commands To Run Project:

1. create a python virtual environment
2. pip install the requirements file given
3. run command: python manage.py runserver

Some of the main features of this project are as follows: 

A. Admin Users Can
-----------------------------------
1. Manage Users (Update, Filter and Delete)
3. Manage ToDo's (View and Process)
